# Geno Schema Language Support for Zed

Adds [Geno](https://crates.io/crates/geno) support to Zed.  

## Features

- Syntax highlighting for all Geno files
- Supports all Geno syntax elements
- Automatically activated for `.geno` files

## Credits

This extension uses the [tree-sitter-geno](https://github.com/jlyonsmith/tree-sitter-geno) parser written by [jlyonsmith](https://github.com/jlyonsmith).
